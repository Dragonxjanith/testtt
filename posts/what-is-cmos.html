CMOS කියන්නෙ මොකක්ද?

මේ CMOS, RTC ( real-time clock ), NVRAM ( non-volatile RAM ) එහෙමත් නැත්තම් CMOS RAM කියලත් හදුන්වනවා. CMOS කියන එකේ තෙරුම complementary metal-oxide semiconductor කියනෙකයි. CMOS කියන්නෙ onboard , battery powered semiconductor chip එකක් computers ඇතුලේ තියෙන information store කරගන්න. Store කරගන්න range එක බැලුවොත් මේක system time සහ Date එකේ ඉදලා system hardware settings දක්වා store කරගන්නවා. Common ම CMOS coin cell battery එක වෙන්නෙ ( Panasonic CR 2023 3V ) කියනෙකයි CMOS memory එක power up කරන්න use කරන්නෙ.

CMOS හි ඉතිහාසය

Motorola 146818 chip එක තමා පලමු RTC සහ CMOS RAM chip එක මුල් IBM computers වල use කරපු. මේකට store කරන්න පුළුවන් උනේ උපරිම Data 64 bytes තමා. System clock එක 14 bytes use කරපු හින්දා system settings store කරන්න තව 50 bytes ඉතුරු උනා.

අද වෙනකොට නම්, computers ගොඩක් settings CMOS එකෙන් move කරලා southbridge හෝ Super I/O chips වලට integrate කරලා තියෙනවා.

මේ CMOS Battery එක කොච්චර කාලයක් පවතිනවද

CMOS battery එහෙක standard life time එක අවුරුදු 10 ක් විතර. නමුත් මේක නම් depend වෙනවා usage එක සහ computer එක තියෙන environment එක අනුව.

CMOS use කරන Devices

  • SRAM ( Static RAM )

අපේ CMOS battery එක fail වෙනවා නම් අපි කොහොමද දැනගන්නෙ

CMOS battery එක fail වෙනවා කියන්නෙ අපේ computer එකට හරියට Date එක සහ Time එක තියාගන්න බෑ.

තව CMOS battery එක fail වෙනකොට අපිට දකින්න ලැබෙනවා indication messages කිහිපයක් ම.

  • CMOS Read Error
