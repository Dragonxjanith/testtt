Google Knowledge Panel එකට සමාන information panel එකක් Facebook එකෙන්

Facebook සමාගම ඔවුන්ගේ users ලට වඩාත් කාර්යක්ෂමව සහ පහසුවෙන් platform එක භාවිතා කිරීම සඳහා අලුත් features සහ services නිරන්තරයෙන් තමන්ගෙ platform එකට එකතු කරනවා. ඉතින් මේ social media දැවැන්තයා ඔවුන්ගේ platform එක හරහා දැනුම වැඩි කරගැනීමට හොඳ අවස්ථාවක් උදා කරමින් තවත් අලුත් revolutionary feature එකක් විදිහට Wikipedia powered knowledge box එකක් announce කරා. ඒක ඔවුන් විසින් හැඳින්වෙන්නෙ Facebook Knowledge Box කියන නමින්. මේක දැනට ක්‍රියාත්මක වෙන්නෙ pilot program එකක් විදිහට සහ දැනට available වෙන්නෙ platform කීපයක විතරයි.

Facebook Knowledge Box එකේ features සහ functions

මේකෙ primary function එක වෙන්නෙ public places, figures වගේ දේවල් ගැන විස්තර user විසින් search කරාට පස්සෙ ඒ අදාල තොරතුරු ලබාදීමයි. මේ නිසා user ට platform එකෙන් පිටතට ගිහින් search කිරීමේ අවශ්‍යතාවය පැන නගින්නේ නෑ. ඒක පහසුවක් විදිහට Facebook ලා පෙන්නලා දෙනවා. ඔවුන් පවසන විදිහට මේ feature එක ක්රියාත්මක වෙන්නෙ regular search bar එකක් හැටියට.



උදාහරණයක් විදිහට user කෙනෙක් යම්කිසි ප්‍රසිද්ධ public place එකක් එහෙමත් නත්තන් public figure එකක් ගැන search කරාම පොඩි knowledge box එකක් දකුනු පැත්තෙන් pop up වෙනවා. ඒ box එකෙන් අර search කරපු දේ ගැන අදාල විස්තර ලබා දෙනවා. දැනට මෙමගින් පෙන්වන්නෙ ඒ ඒ subjects වලට අදාල සීමිත තොරතුරු ප්රමාණයක්. ඒ වගේම මේකෙන් තොරතුරු දෙන්නෙ ප්‍රසිද්ද අය, ස්ථාන ගැන විතරක් නෙමේ මේකෙන් user ට සින්දු, චිත්‍රපටි වගෙ දේවල් ගැන විස්තර ලබා ගැනීමේ හැකියාවත් ලැබෙනවා.

ඒ වගේම තවත් වැදගත් දෙයක් තමා දැනට මේක ක්‍රියාත්මක වෙන්නෙ English language එකෙන් විතරයි. ඉදිරියට අනිත් භාෂා වලින් වගේම හැම user කෙනෙක්ටම මේ feature එක ලැබේවි. ඒත් ඒ කවද්ද කියල නම් තවම Facebook ලා ප්‍රකාශ කරලා නෑ.

ප්‍රකාශ වෙන විදිහට නම් මේ feature එක ඉස්සෙල්ලම ලැබෙන්නෙ desktop, mobile web සහ iOS වලට යි. ඉන් පස්සෙ අපිට මේ feature එක Android app එකටත් ලැබෙයි කියලා බලාපොරොත්තු වෙන්න පුලුවන්. ඒත් ඒක confirm කරන්න හැකියාව ලැබෙන්නෙ Facebook ලා මේ feature එක global market එකට release කරාට පසුවයි.

මේ feature එක introduce කිරීමෙන් Facebook ලා බලාපොරොත්තු වෙන්නෙ users ලට Facebook platform එකෙන් leave වෙන්නෙ නැතුව ඔවුන්ට අවශ්‍ය තොරතුරු ලබා ගන්න උදව් කරන එකයි. මෙතනදි user ගෙ පහසුව වැඩි වෙනවා වගේම platform එක තුලින්ම සිය දැනුම වර්ධනය කර ගැනීමේ හැකියාවත් ලැබෙනවා.
