Android වලින් Internet යන කොට යන Data අඩුකර ගන්නේ කොහමද?

Android Phone එකකින් හෝ Tab එකකින් Internet යන හුගක් අයට තිබෙන ප්‍රශ්නයක් තමයි Data වේගයෙන් ඉවර වීම( Android වලට පමණක් නෙවේ, මෙය ඕනෑම Smart Phone වලට එන OS එකකට පොදු දෙයක්).

Android වලින් Internet යන කොට යන Data අඩුකර ගත හැකි ක්‍රම කිහිපයක් මෙම ලිපියෙන් අපි ඔබට ලබා දෙනවා.

1. Opera Max App එක භාවිතා කිරීම : මිලයන 10 කට වඩා Downloads ඇති මෙම App එකට 4.3 Rating එකක් තිබෙනවා. ඔහුන් මෙම App එක නිර්මාණය කර තිබෙන්න විශේෂයෙන්ම Data Save කර ගැනීම සදහාමයි.

2. Google Chrome’s Data Saver Feature එක භාවිතා කිරීම : හුගක් දෙනා Browser එක භාවිතා කරන්නේ Chrome. ඔබ Opera Max වලට මාරු වෙන්න අකමැති නම් Chrome වල ඇති Data Saver Feature භාවිතා කරන්න. Setting –> Advanced –> Data Server හරහා එය On කරගත හැකිය.

3. Facebook App එක වෙනුවට Facebook Lite App එක භාවිතා කිරීම : Facebook Android App එකත් Android තියෙන හැම කෙනෙක්ම වගේ භාවිතා කරනවානේ. එම App එක සදහාත් Data ගොඩක් වැය වෙනවා. Facebook Lite App එක භාවිතා කිරීම තුලිනුත් එය විශාල වශයෙන් අඩු කරගන්න පුළුවන්.
